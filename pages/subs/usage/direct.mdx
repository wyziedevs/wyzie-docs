### Parameters

| Parameter            | Example                                                | Description                                                                                             |
| -------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------- |
| **id**               | `/search?id=tt3659388` or `/search?id=286217`         | <T id="subs.direct.param.id" />                                                                         |
| **season & episode** | `/search?id=tt0121955&season=1&episode=1`             | <T id="subs.direct.param.seasonEpisode" />                                                              |
| **language**         | `/search?id=tt3659388&language=en,es`                 | <T id="subs.direct.param.language" />                                                                   |
| **format**           | `/search?id=tt3659388&format=srt,ass`                 | <T id="subs.direct.param.format" />                                                                     |
| **hi**               | `/search?id=tt3659388&hi=true`                        | <T id="subs.direct.param.hi" />                                                                         |
| **encoding**         | `/search?id=tt3659388&encoding=utf-8,latin-1`         | <T id="subs.direct.param.encoding" />                                                                   |
| **source**           | `/search?id=tt3659388&source=subdl,podnapisi`         | <T id="subs.direct.param.source" />                                                                     |
| **release**          | `/search?id=tt3659388&release=martian,1080p`          | <T id="subs.direct.param.release" />                                                                    |
| **file/fileName**    | `/search?id=tt3659388&file=proper`                    | <T id="subs.direct.param.file" />                                                                       |
| **origin**           | `/search?id=tt3659388&origin=WEB,BLURAY`              | <T id="subs.direct.param.origin" />                                                                     |
| **key**              | `/search?id=tt3659388&key=YOUR_KEY`                   | <T id="subs.direct.param.key" />                                                                        |
| **refresh**          | `/search?id=tt3659388&refresh=true`                   | <T id="subs.direct.param.refresh" />                                                                    |
